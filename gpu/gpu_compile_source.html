2. Compile g-learn from Source with CUDA#

The second method to make g-learn to be compatible with am existing CUDA installation is to compile imate from source. imate is a dependency package used by g-learn that performs linear algebraic computations on GPU devices. Note that g-learn itself does not have to be compiled from source. See compile imate from the source for details.
